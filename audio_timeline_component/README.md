# Audio Timeline Component

This component powers the interactive audio timeline (WaveSurfer.js) used in the Streamlit demo.

## Build

```bash
cd audio_timeline_component/frontend
npm install
npm run build
```

## Dev server

```bash
cd audio_timeline_component/frontend
npm install
npm run dev
# In another terminal:
# export AUDIO_TIMELINE_DEV_URL=http://localhost:5173
```
